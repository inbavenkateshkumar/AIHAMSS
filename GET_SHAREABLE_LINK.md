# 🔗 Get Your Shareable Link - Step by Step

## Right Now - Get Your Link in 2 Minutes!

### Step 1: Make Sure Server is Running

Open terminal/command prompt and run:
```bash
npm start
```

You should see:
```
🚀 Holy Angels Substitution Server Started Successfully!
Server is running on http://localhost:3000
```

**Keep this terminal window open!**

---

### Step 2: Start Ngrok (New Terminal Window)

**Open a NEW terminal/command prompt window** (keep the first one running!)

**Option A - If ngrok is installed globally:**
```bash
ngrok http 3000
```

**Option B - If ngrok.exe is in your project folder:**
```bash
.\ngrok.exe http 3000
```

**Option C - Use the script:**
- **Windows:** Double-click `start-with-ngrok.bat`
- **Mac/Linux:** Run `./start-with-ngrok.sh`

---

### Step 3: Copy Your Public URL! 🎉

You'll see something like this in the ngrok terminal:


**Copy this URL:**

## 🆘 Troubleshooting

### "Tunnel not found" or Link doesn't work?
- Make sure server is running: `npm start`
- Make sure ngrok is running: `ngrok http 3000`
- Both terminals must be open!

### Friend can't access?
- Check if your computer is online
- Make sure both server and ngrok are running
- Try accessing the URL yourself first

### Need help?
- Check ngrok web interface: http://localhost:4040
- Shows all requests and status
