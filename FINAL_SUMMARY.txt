## 🔧 TROUBLESHOOTING

### "Cannot find module" Error
```
Solution: Run from project directory first
cd C:\Users\Thirumoorthy\Desktop\timetable
node server.js
```

### "Port 3000 already in use"
```
Solution 1: Wait a few seconds and try again
Solution 2: Change PORT in .env file
Solution 3: Kill process using port 3000
netstat -ano | findstr :3000
taskkill /PID <PID> /F
```

### "Database connection failed"
```
Solution: Make sure PostgreSQL is running
1. Start PostgreSQL service
2. Check .env has correct credentials
3. Restart the server
```

### Dashboard looks plain
```
Solution: Refresh browser (Ctrl+F5)
or clear cache and reload
```
